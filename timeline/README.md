## Record Format

각 타임라인 문서는 다음 형식을 사용합니다.

1. 활동 개요
2. 배경 또는 기획 목적
3. 진행 과정
4. 주요 내용
5. 결과 및 후속 조치
6. 공개 가능한 관련 자료

출사·전시·축제처럼 별도 이미지가 있는 활동은 전용 배너를 사용하고, 별도 이미지가 없는 경우 `assets/images/frame-logo.svg`를 기본 배너로 사용합니다.

회원 이름, 학번, 연락처, 계좌 정보, 개인별 투표·평가·보증금 반환 내역 등 내부 민감정보는 공개하지 않습니다.
